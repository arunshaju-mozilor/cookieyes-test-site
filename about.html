About This Site

This site is a controlled test environment for validating the CookieYes cookie consent banner across different pages and user flows.

It is used by the QA team at Mozilor to run automated Playwright tests against real banner behaviour, consent recording, and multi-page navigation.

What we test here

  • Cookie banner display on first visit
  • Accept All / Reject All / Manage Preferences flows
  • Consent persistence across page navigations
  • Banner re-appearance after cookie expiry
  • WIX OAuth signup integration
